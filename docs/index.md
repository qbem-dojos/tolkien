- Modulo: Login Social
    - Facebook
    - Google
    - Github
    - Twitter
    - Discord
    - ICloud
    - Microsoft
- Modulo: Criação do Personagem
    - Submodulo: Criação do Avatar
    - Submodulo Character Sheet
- Modulo: Criação de Campanhas
