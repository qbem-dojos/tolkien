# Tolkien
Plataforma open source para jogar RPG de mesa com os amigos.

## Funcionalidades
- Módulo: Login Social
    - Facebook
    - Google
    - Github
    - Twitter
    - Discord
    - ICloud
    - Microsoft
- Módulo: Criação do Personagem
    - Submodulo: Criação do Avatar
    - Submodulo Character Sheet
- Módulo: Criação de Campanhas
